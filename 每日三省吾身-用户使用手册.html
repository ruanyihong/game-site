快速开始

GET STARTED

不需要注册账号、不需要密码。打开网页就能用，几秒上手。

打开网站
在电脑或手机浏览器访问 https://review.ruanxunan.top，即可进入主页面。
直接开始记录
不登录也能用：点击「＋ 新建任务」即可添加你的第一个计划。
（推荐）开启云同步
点右上角「☁ 云同步」→「创建我的云空间」，得到一个 6 位空间码。这样数据会存到服务器，换设备也不丢。
💡 小提示第一次用，建议先点「云同步」创建一个空间，再开始记任务——省得以后手动搬数据。

核心功能一览

FEATURES

本工具围绕「计划 → 执行 → 复盘」的闭环设计，共 6 大模块：

📋 任务看板
规划每周 / 每日任务，拖拽管理进度，支持四种可视化视图。
🔥 习惯养成
建立每日习惯，一键打卡，生成连续天数与热力图。
🍅 番茄钟
25/45 分钟专注计时，绑定任务，记录专注时长。
📝 每日 / 每周复盘
自动生成复盘卡片，反思「三省」，沉淀成长记录。
☁ 云同步
用空间码跨设备同步，或分享给别人共用同一份计划。
💾 数据备份
一键导出 / 导入 JSON，本地留底，万无一失。

任务看板与四种视图

KANBAN & VIEWS

顶部有 4 个标签页，切换不同视角看同一批任务：

视图适合做什么
看板按「待办 / 进行中 / 已完成」分列，拖拽卡片管理进度，最常用。
日历按日期排布任务，直观看到某天安排了什么。
时间线横向时间轴，看任务在时间上的先后与重叠。
甘特图项目式排期，看清任务工期与依赖关系。
新建任务
点击右下角浮动按钮 ＋ 或「＋ 新建任务」，填写标题、截止日、分类、子任务（最多 5 个）。
管理任务
点击卡片查看详情、编辑、删除；在看板里拖拽卡片改变状态。
连续记录
点「🔥 连续」按钮查看你的打卡 streak，坚持天数一目了然。

习惯养成与连续打卡

HABITS

点击顶栏「追踪习惯」进入习惯页（habits.html）。

添加习惯
设定一个你想每天坚持的事，例如「阅读 30 分钟」「背单词」。
每日打卡
完成就点一下，记录今天已做；连续打卡天数会自动累加。
查看热力墙
页面展示按日期的打卡热力图，坚持越久颜色越亮，成就感拉满。
📌 关于连续性漏打卡当天会中断连续计数（按自然日）。但历史记录都保留，可随时回看。

番茄钟专注计时

POMODORO

点顶栏「🍅 番茄钟」打开计时面板。

选择时长
支持 25 分钟（标准番茄）与 45 分钟（深度专注）两种模式。
绑定任务（可选）
下拉选择一个看板任务，计时结束后专注时长会记到该任务上。
开始 / 重置
点「开始」倒计时，「重置」归零。专注数据存本地，开启云同步后可跨设备累计。

每日 / 每周复盘卡

REVIEW

点顶栏「📝 复盘」打开复盘卡片，这是「每日三省吾身」的精华所在。

🌅 每日复盘
每天结束时，回答「今日得失 / 明日改进 / 一句感悟」，系统自动按日期存档。
🗓️ 每周复盘
周末回顾本周计划完成率与习惯坚持，沉淀成周记，便于长期追踪。
✍️ 建议养成「每天 2 分钟写每日复盘」的习惯，半年后回看会非常有价值。

云同步与分享（重点）

CLOUD SYNC & SHARE

这是云同步版和传统单机版最大的区别：数据存在服务器上，可跨设备、可分享。

① 创建你自己的空间

点「☁ 云同步」
在右上角，点击云图标打开同步面板。
「＋ 创建我的云空间」
系统生成 6 位空间码（如 K7M2QP），并把当前数据上传到服务器。
之后自动同步
你每次改动，页面会在 1~2 秒内自动保存到服务器，无需手动操作。

② 换设备 / 换浏览器找回数据

在新设备的浏览器打开 review.ruanxunan.top，点「☁ 云同步」→ 输入同一个空间码「加入」，数据立刻回来。空间码存在你常用浏览器的本地，同一个浏览器下次打开会自动同步，不用重输。

③ 分享给别人共用

复制空间码或链接
在同步面板点「复制空间码」或「复制分享链接」。
发给朋友
对方打开网站 →「☁ 云同步」→ 输入码加入，即可看到并编辑同一份计划。
⚠️ 多人协作须知同一空间采用「最后保存者覆盖」策略——适合个人多端同步、或小范围协作。多人同时大改同一份数据时，后保存的会覆盖先保存的，注意错峰编辑。
🔗 分享链接链接形如 https://review.ruanxunan.top/?join=K7M2QP，对方点开会自动填入空间码并加入，最省事。

数据备份与迁移

BACKUP

虽然数据已在服务器，仍建议定期本地备份一份，双重保险。

导出
点顶栏「💾 数据」→ 导出，浏览器会下载一个 .json 文件，存到电脑即可。
导入
换电脑或数据异常时，点「💾 数据」→ 导入，选择之前的 json 文件恢复。
退出同步
在云同步面板点「退出同步」，本机数据保留，只是停止与服务器同步。
🗂️ 服务器端备份所有数据在服务器 /opt/daily-review/data/ 目录（每个空间一个 json 文件）。运维者可定期拷贝该目录做整机备份。

常见问题 FAQ

FAQ
关掉浏览器后数据会没了吗？
不会。数据存在浏览器本地（localStorage），关掉再打开同一浏览器还在。若已创建云空间，空间码也记在本地，下次打开会自动同步，无需重输。
换手机 / 换电脑怎么拿回数据？
打开网站 →「☁ 云同步」→ 输入你的 6 位空间码加入，数据立刻从服务器拉回。
需要注册账号吗？
完全不需要。用「空间码」机制代替账号，免注册、免密码、即开即用。
分享给别人后，他能改我的数据吗？
能。同一空间码下所有人读写同一份数据（最后保存者覆盖）。只自己用就不分享即可。
忘记空间码了怎么办？
常用浏览器里仍记着码，打开同步面板即可看到。若清了缓存，可用「💾 数据」导入本地 json 恢复，或重新创建新空间。
网站打不开 / 很慢？
检查网络；或确认服务器正常运行。若长时间不可用，可用本地导出的 json 先在别处恢复查看。
数据会丢吗？
三重保障：①浏览器本地 ②服务器 ③你手动导出的 json。同时丢的概率极低。
