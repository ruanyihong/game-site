# workbuddy 初认识 · 课0（网页版课件）

一份单文件网页课件，适配电脑/手机浏览器，含 20 页内容与 16 张配图。

## 本地预览
直接双击 `index.html` 即可在浏览器打开（联网时会自动加载字体）。

## 部署
本文件夹已配置好 GitHub Pages 所需的一切（含 `.nojekyll`）。
按 `部署教程.md` 操作即可发布到 `https://<你的用户名>.github.io/<仓库名>/`。

## 目录
- `index.html` — 课件主体（CSS/JS 内联，无需任何依赖）
- `assets/` — 16 张页面配图
- `.nojekyll` — 让 GitHub 原样托管，不做额外处理
