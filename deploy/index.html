P04 · 为什么先「认识」再「用」

终局，是带它走出去交付价值

这套训练，终局是让你们带着 WorkBuddy 走出去，给真实的商铺、真实的人交付价值。但前提——工具你先得玩明白。

我见过太多人第一次打开就卡在「登录不上」「找不到对话框」「选错模式白干半小时」。今天这一课，就是把所有「第一次的坑」提前替你踩完。

工具顺手，才能把脑子用在真正重要的事上。

P09 · 界面 · 中间

用自然语言下指令

  • 可以打字，也能传文件、贴图片、发链接当上下文。
  • 回复时你会看到它的思考过程与中间步骤——别慌，它在「干活」，不是卡了。
  • 想纠正就接着说，像跟同事聊天一样。
  • 多说一句：说清背景 + 目标 + 约束，结果天差地别（下一课专门讲）。
练

破除陌生感：在对话区打一句「你好，你是谁」，看它怎么回。

真实对话区：输入框 + 上下文 + 工作空间选择。

↳ 会下指令，才敢跟客户现场演示

P12 · 个性化设置 · 细节提示

几个我强烈建议现在就动的

  • 通知与隐私——决定远程时它能否给你推消息。
  • 记忆开关——是否允许它记住你的偏好（下一站细讲）。
  • 默认输出语言 / 格式——常交中文稿，就别让它默认英文。

这些不是炫技，是让你少说废话、它少猜你的意思。

↳ 早设早省心，少说废话多干活

P13 · 板块四 · 3 大工作模式

三种「工作性格」，对应不同任务

CRAFT
你说我做

你下指令，它立刻执行。适合目标清楚的小事。

PLAN
先规划后执行

先出方案给你看，你认可了再干。适合复杂多步任务。

ASK
只问不做

它只分析、不改动。适合只想听建议、不敢让它动时。

选对模式，效率翻倍；选错，要么白等，要么白干。

↳ 选对模式效率翻倍，选错白等白干

P14 · 模式一 · Craft
CRAFT
你说我做
最常用默认模式。给一句清楚的话，它直接开干、给结果。
示例：把这段文案改成朋友圈风格 示例：生成一张咖啡店开业海报

适合你已经想明白要什么。若自己都模糊，用它易「做完了但不是你要的」。

练

实操：用 Craft 模式发一句清晰指令（如改文案 / 生成海报），看它秒回。

实操：一句话说清，Craft 直接开干
↳ 小事立刻干，别把时间耗在犹豫

P16 · 模式三 · Ask
ASK
只问不做
最特别——只分析、不改动任何东西。给判断，但绝不替你改文件、不替你发消息。

只读 · 无副作用

「这份合同有没有风险」「这个方案行不行」——它给判断，绝不动手。

安全顾问

改老板文档前，先问 Ask 一遍「哪里要改」，心里有数了再换 Craft 去改。

当你还不敢让它动手的时候，Ask 是你的安全选项。

只问不做：先把问题想透、问准
↳ 不敢动时，有安全顾问兜底

P17 · 3 大模式 · 怎么选

任务来了，点哪个？

TASK IN
任务来了
先问自己：想清楚了吗？事大吗？敢让它动吗？
Craft
想清楚了要什么 → 直接干，立刻执行。
Plan
事大、步骤多、怕跑偏 → 先对齐，出方案再跑。
Ask
只想听判断、不敢让它动 → 只问不做。
拿不准？默认从 Plan 起步最稳——最多多花两分钟看方案，绝不做错方向。
